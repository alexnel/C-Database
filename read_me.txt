To run file, go to directory of file in your terminal and type 'make', once projected has compiled, type 'make run' to start the program. Follow the on-screen promps to manipulate the database as required. All functionality should be working fully.

If you wish to load data from a file to the database, ensure that the file is in the following format:
Name Surname StudentNumber Classrecord(list of marks separated by spaces)
Each entry should be on a new line and each element should be separed by a space. If the persons name has multiple words, link works with underscores, e.g. Surname: 'Van Zyl' should be entered as 'Van_Zyl'.
